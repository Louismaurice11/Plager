How it Works

This algorithm works by dividing the text into overlapping windows of fixed length w. Within each window, the algorithm identifies the hash value of the minimum-weighted element based on a predefined weighting scheme. This hash value is usually referred to as the fingerprint of the window. The fingerprint of each window is then compared with its neighbours to identify the strongest matches.
